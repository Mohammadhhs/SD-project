# System Dynamics Modeling and Simulation

This repository contains the project files, models, and documentation for a comprehensive System Dynamics simulation conducted as part of a graduate/undergraduate course at the Department of Management and Economics, Sharif University of Technology.

## Project Overview
The project focuses on system thinking and mathematical modeling to analyze complex, non-linear, and feedback-driven socio-economic or managerial challenges. 
* **Supervisors:** Dr. Alinaghi Mashayekhi & Dr. Vahid Zahedi
* **Focus Area:** [e.g., Human Capital Migration / Workforce Dynamics / Supply Chain Optimization]

## Key Methodologies & Skills Applied
* **Systems Thinking:** Developing Causal Loop Diagrams (CLDs) to map complex feedback loops and systemic interdependencies.
* **Mathematical Modeling:** Constructing Stock and Flow diagrams to simulate dynamic behaviors over time.
* **Scenario Analysis & Policy Design:** Utilizing simulation tools (like Vensim/iThink) to evaluate different policy scenarios and design optimal, long-term managerial interventions.
